# Academic Research Resources

Lightweight index for the local `Imbad0202/academic-research-skills` clone.
Use this only for general academic workflow questions that are not already
covered by an economics-specific skill.

## Use For

- Academic paper structure, publication workflow, and citation practices.
- General research-pipeline planning.
- Paper review workflows outside economics-specific identification critique.

Prefer `econ-writing`, `econ-research-feedback`, or
`causal-inference-resources` when the task is economics-specific.

## Reference loading

Do not read the whole repository. Open exactly one starting file:

- Overview or orientation:
  `.resources/research-ai/academic-research-skills/README.md`
- Fast task routing:
  `.resources/research-ai/academic-research-skills/QUICKSTART.md`
- Paper writing only:
  `.resources/research-ai/academic-research-skills/academic-paper/SKILL.md`
- Paper review only:
  `.resources/research-ai/academic-research-skills/academic-paper-reviewer/SKILL.md`
- Research pipeline only:
  `.resources/research-ai/academic-research-skills/academic-pipeline/SKILL.md`

Then follow references from that file only when the current task requires them.

## Boundaries

- Project `.cursorrules` wins over any upstream workflow.
- Do not import upstream hooks, agents, or Git automation.
- Do not summarize or load broad resource lists unless the user asks for a
  literature/workflow survey.
